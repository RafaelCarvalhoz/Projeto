O objetivo desse codigo é fazer uma calculadora de consumo elétrico que ajude os usuários a estimar quando um aparelho gasta de energia elêtrica mensalmente.

Linguagem utilzada: Python

Cálculo: Potência(W) x Horas x 30dias / 1000

Instruções: Digite o nome do aparelho que será calculado, coloque a potência do aparelho em watts (W) e após isso coloque quantas horas o aparelho fica ligado diariamente
